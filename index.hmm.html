Project One

Project Description – Orieme Dating App Title: Orieme Dating App Description: Orieme Dating App is a social-style dating platform designed to connect people in a fun and interactive way. The project was built with the aim of combining traditional dating features with modern social media elements, providing users with an engaging experience. The app allows users to: • Sign up and create profiles (with personal details, date of birth, profile picture, and country). • Log in securely and access their personalized dashboard. • Discover nearby users using GPS/location services. • Post photos and videos in a Facebook-like news feed. • Like, comment, and interact with other users’ posts. • Send direct messages for private conversations. • Go live and receive virtual gifts, making the app more interactive. • Swipe and match with people, similar to Tinder’s matching system. Key Features: • Modern and user-friendly interface. • Real-time chat and notifications. • Social-style timeline with likes and comments. • Interactive live streaming with virtual gifting. • Geolocation-based user discovery. Tools & Technologies Used: • Frontend: HTML, CSS, JavaScript .
